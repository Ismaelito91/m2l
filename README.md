# Maison des ligues 

site de E-commerce qui vend des produits en lien avec le sport

## Installation

```bash
git clone https://github.com/Ismaelito91/m2l.git
```
pour le front
```bash
cd front 
npm install
npm init
````
Pour le back
````bash
cd back
npm install
npm init
````
## Lancement de l'app
pour le front :
````bash
cd front
npm run dev  
````

Pour le back : 
````bash
cd back 
npm run  nodemon index.js
````
## BDD
importer dans MYSQL le fichier M2l.sql qui est dans le repo git
